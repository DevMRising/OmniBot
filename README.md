# Имитационная модель Omnibot в робототехническом симуляторе CoppeliaSim
Задание:

    1. Используя CAD-модель Omnibot сконструировать имитационную модель в среде CoppeliaSim;
    2. С помощью элемента Path в среде CoppeliaSim создать путь движение мобильного робота;
    3. Разработать программное обеспечение имитационной модели Omnibot.
